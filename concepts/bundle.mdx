## Explanations of some fields

| Name         | Explanation |
| ---------    | ----------- |
| Generate a certificate    | If set to yes, a certificate for the bundle will be accessible to a user who have completed each product of the bundle. |
| Contact person    | If a user is selected, the Question or comment form at the right of the bundle's catalog page will be sent to that user. If no user is selected, the form will be set to the Contact email in the Email tab of the Settings page.  |
| Added to home spotlight?    | If set to yes, and an Image spotlight has been uploaded, the image will be adedd to the home slideshow.|
