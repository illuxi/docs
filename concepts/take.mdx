---
title: 'Enrollment Record'
description: "An Enrollment Record tracks a user’s progress in a specific product. It includes key information such as the start date, learning progress, completion status, and any results (like scores or certificates). It represents the user actually taking the product, not just having access to it."
icon: 'poll'
---

## Status

An access has different status based on its lifecycle.

| Status                     | Description                                                                                  |
|--------------------------------------|----------------------------------------------------------------------------------------------|
| In Progress                | The user has started the product, but has not completed it yet|
| Done   | The user has completed the product.      |
