## Capsule types

Here are the different types of capsule.

| Type         | Description |
| ---------    | ----------- |
| Pre-scheduled exam    | An exam is a set of questions grouped in sections ans subsections. Exams are done at a predefined date and time with a set duration. Every learner proceed to the exam at the same time. |
| Quiz    | A set of questions with correct answers. |
| SCORM xAPI cmi5 HTML5 | A learning module created via one of the affordmentionned technologies.|
| Self-service exam   | Set of questions grouped in sections ans subsections. |
| Synchronous activity  | A synchronous activity happening at a specific date and time, online or in person.       |
| Text / file  | Text content and/or an embeded PDF file or web link. |
| Video | An uploaded video or a video available on Vimeo or Youtube. |
