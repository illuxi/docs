## Capsule types

Here are the different types of capsule.

| Type | Description |
| --- | --- |
| Pre-scheduled exam | An exam is a set of questions organized into sections and <br />subsections. <br />Exams are scheduled for a specific date and time<br />and must be completed within a set duration. <br />All learners take the exam at the same time. |
| Quiz | A set of questions with predefined correct answers. |
| SCORM xAPI cmi5 HTML5 | **A learning module created using one of the aforementioned**<br />**technologies.** |
| Self-service exam | A set of questions grouped into sections and subsections. |
| Synchronous activity | A synchronous activity scheduled for a specific date and time, <br />delivered online or in person. |
| Text / file | Text content and/or an embedded PDF file or web link. |
| Video | An uploaded video or a video hosted on Vimeo or YouTube. |
