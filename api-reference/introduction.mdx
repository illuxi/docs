## Authentication

With every request, a token will need to be included in the header.

To obtain a valid token, you will need the URL of your portal's API as well as your API key. You can find both information in your portal’s admin under Apps / illuxi API.

![API access info](/images/api-access-info.png)

Use the [/token](api-reference/endpoint/token) endpoint to obtain a valid token.

## Autorisation header

Each request needs to contain this specific header:

```curl
Authorization: <TOKEN>
```
